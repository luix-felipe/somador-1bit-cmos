# Somador Completo 1 Bit — Mirror Adder CMOS 0,12µm

Projeto final — Capacitação em Microeletrônica | CI Amazônia    

## Topologia
Mirror Adder full-custom, 28 transistores, tecnologia 0,12µm, Vdd=1,2V

## Estrutura
- `verilog/` — descrição HDL das células e do somador
- `dsch/`    — arquivos de esquemático DSCH2
- `microwind/` — arquivo de layout MICROWIND
- `sim/`     — formas de onda e resultados de simulação
- `relatorio/` — Technical Paper final
- `docs/`    — referências e anotações
